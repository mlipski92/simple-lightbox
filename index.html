Hello! This is simple gallery!



To include this gallery to your website you have to enter DIV element with class "simpleJsGallery" and your unique gallery id. Inside you have to insert A items with class "simpleJsGallery__item" and img inside.
Those items are inserted correctly.
  • If the first element of A item with "simpleJsGallery__item" class isn't IMG this item is interpretted as broken. Example:
    Hello!


If you want you can create new gallery, but remember to enter an unique ID!


The last thing you have to do is include JavaScript and CSS files from /assets and list your galleries in script in tags SCRIPT
Example:
const firstGallery = new simpleGallery('firstGallery');
